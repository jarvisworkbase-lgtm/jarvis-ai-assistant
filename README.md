# 🤖 Jarvis - Local AI Assistant

A locally running AI assistant built with Python and Ollama. 
Jarvis can answer questions, hold conversations, and runs 
completely offline on your PC — no internet or paid API needed.

## 🚀 Features
- 💬 Chat with an AI locally on your machine
- 🔒 100% private — no data sent to the cloud
- ⚡ Fast responses using Ollama's local LLM engine
- 🐍 Built with Python

## 🛠️ Technologies Used
- Python
- Ollama
- Local LLM (e.g., LLaMA / Mistral)

## 📦 How to Run
1. Install [Ollama](https://ollama.ai)
2. Clone this repository
3. Run `python jarvis.py`
